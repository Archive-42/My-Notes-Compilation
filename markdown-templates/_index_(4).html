The Typescale

Desktop and tablet typescale is based on major third typescale with a base of 100% of base font size set by the user which is typically 16-18px. On mobile phones the typescale is based on a major second typescale with a base of 100% as set by the user.

ZillaSlab highlight

Any heading level can be set to use the ZillaSlab Highlight style by adding the class slab-highlight, for example: 

<h2 class="slab-highlight">Slab Highlighted</h2>
