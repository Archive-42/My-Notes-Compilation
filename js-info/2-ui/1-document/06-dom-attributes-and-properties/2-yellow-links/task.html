Make external links orange

Make all external links orange by altering their style property.

A link is external if: - Its href has :// in it - But doesn’t start with http://internal.com.

Example:

```html run the list
  • http://google.com
  • /tutorial.html
  • local/path
  • ftp://ftp.com/my.zip
  • http://nodejs.org
  • http://internal.com/test

```

The result should be:

[iframe border=1 height=180 src=“solution”]
