## The postfix and prefix forms

<span class="task__importance" title="How important is the task, from 1 to 5">importance: 5</span>

What are the final values of all variables `a`, `b`, `c` and `d` after the code below?

    let a = 1, b = 1;

    let c = ++a; // ?
    let d = b++; // ?

solution

The answer is:

- `a = 2`
- `b = 2`
- `c = 2`
- `d = 1`
